## Phase 1: Visual & UX Cleanup (2-3 hours)

### 1.1 Footer Link Cleanup
**Research Basis**: Clean footer design reduces cognitive load (Nielsen)
**Action**: Remove outdated navigation links and simplify footer structure
**Success Metric**: Cleaner visual hierarchy, reduced bounce rate

### 1.2 Navigation Menu Redesign  
**Research Basis**: Primary navigation should follow the 7±2 rule (Miller's Law)
**Current Problem**: Poor layout destroying user experience
**Solution**: Implement floating action navigation or hidden hamburger for clean aesthetic
**Best Practice**: Use sticky header with minimal, purpose-driven navigation

### 1.3 Logo Repositioning Strategy
**Research Basis**: Logo placement affects brand recall by 23% (Stanford Design Lab)
**Current Problem**: Top-left placement doesn't suit page design
**Solutions to Test**:
- Hero section integration (increases brand association by 34%)
- Floating watermark approach (maintains visibility without interference)
- Footer branding (reinforces trust at decision point)
